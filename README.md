🌑 DARKSKY

Hybrid SDR Companion for SDRconnect via WebSocket & RTL‑SDR USB
DARKSKY is a lightweight hybrid companion tool designed to extend SDRconnect using a WebSocket bridge and optional RTL‑SDR USB integration.
It provides a simple, responsive interface for interacting with SDR hardware, visualising data, and controlling key functions without the overhead of a full SDR suite.
---
🚀 Features

• WebSocket bridge for communicating with SDRconnect
• RTL‑SDR USB support for direct hardware access
• Hybrid HTML/Python interface
• Lightweight, portable, no installation required
• Includes full user manual (PDF)
• Open for modification and personal use
• Non‑commercial licence (CC BY‑NC 4.0)

🛠️ Installation

  1. Install Python dependencies
From the project directory:

pip install -r requirements.txt

  2. Connect your SDR hardware
   
• SDRconnect running with WebSocket support enabled
• Optional RTL‑SDR USB device connected to your system

  3. Launch DARKSKY
Run the Python companion:

python "DARKSKY v1.2.0 – Hybrid SDR Companion.py"

  4. Then open the HTML interface in your browser if required:

DARKSKY.html
