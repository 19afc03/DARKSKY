Schedule Database Sources
EIBI and AOKI are downloaded automatically. HFCC and user CSV are loaded from files placed in the same folder as darksky_server.py.
HFCC schedule — download from hfcc.org/data and save as hfcc_sked.csv alongside the server:
User schedule CSV — your own frequencies. Columns: freq_mhz (or freq_khz), name, type, schedule, location, notes. Comma or semicolon delimited, with header row.
FMLIST — download from fmscan.org/tools.php → Userlists → FM. Save as fmlist_sked.csv:
MWLIST — same site, select MW/LW tab. Save as mwlist_sked.csv:
Files are sent to the DARKSKY server and saved alongside darksky_server.py. The server reloads the database automatically after saving.
